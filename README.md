# 센스퀘어 (Sensequare)

아이티센 13기 7조 팀 프로젝트  
카카오 로그인 기반의 아이티센 커뮤니티 & 근무지 시각화 플랫폼

---

## 🔍 주요 기능 요약

### 👥 회원
- 카카오 소셜 로그인 및 회원가입
- 명함 OCR을 통한 사내 인증
- 사용자 정보 조회 및 수정

### 📝 게시판
- 게시글 작성/수정/삭제/임시저장/복구
- 좋아요 및 인기글 기능
- 댓글/대댓글 기능
- 키워드 기반 검색 및 자동완성
- 실시간 인기 검색어 제공

### 🗺️ 근무지 지도
- 근무지 위치 시각화  
- 법인별 마커 색상
- 필터링 및 검색 기능
- 리스트-지도 연동

### 💬 채팅
- 사용자 목록 조회
- 메시지 송수신

---
